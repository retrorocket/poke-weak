## 概要

* ポケモンの相性チェック用LINE Botです
  - LINEに防御側のタイプ名を投稿すると、攻撃側の相性を返却します。
* ポケモンの名前から相性を出すのは対応してないです
  - [fanzeyi/pokemon\.json: Pokemon dataset in JSON\.](https://github.com/fanzeyi/pokemon.json) が8世代に対応したらやるかもしれない
* 日本語のみ対応
